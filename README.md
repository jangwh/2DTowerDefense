## 1. 프로젝트 개요

좀비vs식물을 레퍼런스로 제작한 모작입니다.

Unity를 활용하여 2D로 제작하였습니다

개발기간 : 2025.07.07 ~ 2525.07.18

## 2. 주요 기능
   
2.1 드래그 앤 드롭
* 설치할 타워를 클릭 후 드래그
* 원하는 위치에 드랍 할 시 타워가 설치

2.2 오브젝트 풀링
* LeanPool을 활용하여 오브젝트 풀링 구현

2.3 데이터관리
*Json을 활용하여 데이터를 관리
