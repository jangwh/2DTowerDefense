## 1. 프로젝트 개요

좀비vs식물을 레퍼런스로 제작한 모작입니다.

Unity를 활용하여 2D로 제작하였습니다

개발기간 : 2025.07.07 ~ 2525.07.18

## 2. 주요 기능
   
2.1 게임시스템
* 게임 플로우
* 드래그 앤 드롭
* 오브젝트 풀링
* UI

2.2 데이터
* PlayerPrefs
* Json

2.3 상점
* 타워 구매, 강화
